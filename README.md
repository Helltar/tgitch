## Tgitch

Simple code for a simple bot which sends messages from selected **Twitch**-chat to your **Telegram**-channel.

Modify three files:

```
telegram_channel.txt
twitch_channel.txt
twitch_username.txt
```
... and create two files with you tokens:

```
telegram_token.txt
twitch_oauth.txt
```
**Done.**
