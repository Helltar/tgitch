## Tgitch

A simple code for creating simple bot for send messages from your **Twitch** chat to you channel in **Telegram**

Modify two files:
```
telegram_channel.txt
twitch_username.txt
```
... and create two files with you tokens:
```
telegram_token.txt
twitch_oauth.txt
```
**Done.**
